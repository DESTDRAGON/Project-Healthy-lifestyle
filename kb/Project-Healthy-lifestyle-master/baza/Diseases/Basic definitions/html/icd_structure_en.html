The full ICD-11 document includes an introduction, classification, instructions for completing a certificate of perinatal mortality, regulatory definitions, nomenclature provisions and other sections. We describe the structure and principles of classification.

In the classification of the ICD-11, the statistics are grouped into the following groups:
-Epidemic diseases;
-constitutional or common diseases;
-local diseases grouped by anatomical localization;
-developmental diseases;
-injuries.

The classification has a four-level hierarchical structure. Records classification have codes and verbal transcript. The classification uses an alphanumeric coding system. The structure and classification codes are as follows:

Classes

Only 27 classes.
The code of each class is a pair of codes defining the range of codes included in the class.
Class code is presented in the form: letter + two numbers - letter + two numbers. For example: A00-B99, C00-D48, K00-K93.

Blocks Rubrics

Classes ICD-11 are divided into homogeneous blocks. Block code in the form of a letter + two digits or through an indication of the range. For example: B99, C00-C97, K00-K14.
Three-digit headings

View code: letter + two numbers. For example: C02, K00.
Four-digit subheadings

Subcategory code in the form: letter + two numbers. numeral. The code uses the point after which there is a figure specifying the disease for a three-digit heading. For example: C02.1, K00.3.
